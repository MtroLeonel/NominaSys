## 📊 Resumen Ejecutivo del Sistema

### ¿Qué hace este sistema?

Sistema web de gestión de nómina empresarial que permite:
- ✅ Administrar empleados, departamentos y puestos
- ✅ Calcular nóminas (semanal, quincenal, mensual)
- ✅ Aplicar bonos automáticamente
- ✅ Registrar cambios en bitácora
- ✅ Soporte para MySQL y SQL Server

### Tecnologías

- **Backend:** Node.js + Express.js 5
- **Base de Datos:** MySQL / SQL Server (Sequelize ORM)
- **Frontend:** EJS + Bulma CSS
- **Patrón:** MVC (Model-View-Controller)

### Características Clave

- 🔄 Multi-base de datos (MySQL/MSSQL)
- 🎨 Interfaz amigable y responsive
- 📊 Cálculo automático de nóminas
- 🎁 Sistema flexible de bonos
- 📝 Auditoría completa con bitácora
- 🚀 Fácil instalación y configuración

## 🛠️ Comandos Rápidos

```bash
# Instalación
npm install

# Configurar base de datos (editar .env)
copy .env.example .env

# Inicializar con datos
npm run seed

# Desarrollo
npm run dev

# Producción
npm start
```
